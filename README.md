# Website Management-Products

Ini adalah website Hospital-Patient-Data, di web ini kita bisa melakukan seperti Menambah, mengedit, dan menghapus data pasien.
Melihat informasi terperinci tentang data pasien. aplikasi ini juga sudah terintegrasi dengan database MYSQL

# Requirement

Buatlah Database dengan nama : `hospital`

Buatlah Tabel : `pasien`
